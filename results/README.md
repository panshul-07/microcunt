# Results Folder

This folder stores run outputs and observations.

Generated by training script:
- `training_history.csv`
- `training_metrics.json`
- `training_observations.md`

Generated by conversion script:
- `conversion_report.json`
- `conversion_observations.md`

Generated by plotting script:
- `plot_training_curves.png`
- `plot_model_quality_size.png`

Run:

```bash
python3 python/train_tinyml_model.py
python3 python/convert_to_tflite.py
python3 python/generate_result_plots.py
```
